寻觅梧州踪迹

梁羽生公园位于广西梧州市蒙山县。国家4A旅游景区。该地原名鳌山公园，因形似巨鳌而得名，始建于1762年，一代武侠小说大师梁羽生少年时代曾常在此处游玩。
龙母庙位于广西梧州市万秀区。国家4A旅游景区”梧州骑楼城—龙母庙景区”的核心景点。据中国当代著名国学家、文化学者肖健介绍
骑楼城位于广西梧州市万秀区。国家4A旅游景区”梧州骑楼城—龙母庙景区”的核心景点。现存骑楼街道22条，总长7公里，最长街道达2530米，骑楼建筑560幢，规模之大、数量之多，国内罕见，被誉为”世界第八奇迹”。
梧州园博园位于广西梧州市龙圩区。国家4A旅游景区”梧州苍海旅游区”的主要景区之一。
梧州园博园位于广西梧州市龙圩区。国家4A旅游景区”梧州苍海旅游区”的主要景区之一。
永安王城位于广西梧州市蒙山县。国家4A旅游景区。蒙山是天朝封王地，秀美长寿乡，是一个旅游度假养生的好地方。蒙山蕴含丰富的历史文化底蕴、自然风景秀丽、人文风情浓厚
石表山休闲旅游风景区位于广西梧州市藤县境内。国家4A旅游景区。景区地处珠三角经济圈和北部湾经济圈的交汇点，交通极为便利。
天龙顶山地公园位于广西梧州市岑溪市。为国家4A旅游景区，是中国第一个全时度假山地户外休闲景区。
